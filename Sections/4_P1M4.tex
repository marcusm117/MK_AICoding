\section{Market 4: Vibe Coding Tools}

\begin{frame}{Market 4: Scope \& Definition}
\justifying
\small
\begin{itemize}
  \item \textbf{Research Object:} Natural language-to-application platforms enabling non-programmers to build functional software---``vibe coding'' tools that abstract away traditional development workflows.

  \item \textbf{Key Players:}
  \begin{enumerate}
    \item \textbf{Web App Builders:} Bolt, Lovable, v0 (Vercel), Create.xyz.
    \item \textbf{Full-Stack Platforms:} Replit Agent, Glide, Softr.
    \item \textbf{Enterprise/Internal Tools:} Retool AI, Airplane, Superblocks.
  \end{enumerate}

  \item \textbf{Core Capability Assessment:} Four capabilities define the vibe coding value chain:
  \begin{enumerate}
    \item \textbf{User Intent Expansion} $\rightarrow$ transform brief, incomplete user descriptions into full implementations---either by asking targeted clarifying questions (without overwhelming the user) or by inferring intent and making reasonable product decisions.
    \item \textbf{Conversational Editing} $\rightarrow$ refine and extend applications through dialogue with natural language, which is the most natural interface for users.
    \item \textbf{Visual Precision Editing} $\rightarrow$ non-technical users can make precise adjustments via visual editors, drag-and-drop, or direct manipulation beyond the limitations of natural language.
    \item \textbf{Deployment Automation} $\rightarrow$ one-click deployment, hosting, and scaling without DevOps knowledge.
  \end{enumerate}
\end{itemize}
\end{frame}
